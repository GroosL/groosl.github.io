Some of the tools and applications I daily drive

I run a self built desktop tower and a standard Thinkpad T480

PC Hardware

  • Ryzen 7 5700
  • RX 7600
  • 32GB 2667Mhz
  • Gigabyte B550M-K

Software

  • Both my machines run NixOS. Both utilizing my hand made flake hosted on my private git server so that they can be easily and safely managed and updated
  • My WM of choice is my custom build of dwl
  • My C/C++ compiler of choice is clang
  • Foot is my terminal
  • And bemenu is my applauncher/menu app
  • I use Brave and Librewolf as my web browsers, with Brave bein the main one
  • I do all of my coding on a neovim's distro called NvChad
  • My shell is a heavily configured ZSH
  • Gopass is my password manager
  • I mostly try to use functional software, usually lightweight and prefer CLI apps
